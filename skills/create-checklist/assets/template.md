## 動作確認チェックリスト

各項目には対応する AC 番号を `[ACn]` 形式で併記する。AC に紐付かない補助項目は `[補助]` を付ける。

### 正常系

- [ ] [AC2] 正常なメールアドレスを入力するとエラーが表示されない

### 異常系

- [ ] [AC1] `invalid` のような形式不正なメールアドレスでエラーメッセージが表示される
- [ ] [AC1] `@example.com` のように local part が無いメールでエラーメッセージが表示される
- [ ] [AC3] メールアドレス未入力で送信時に必須項目エラーが表示される

### エッジケース

- [ ] [AC1] 非常に長い文字列（256 文字超）でエラーメッセージが表示される
- [ ] [AC2] 国際化ドメイン（`user@例.jp`）が許容される

### 補助項目

- [ ] [補助] 入力中の他フィールドへのフォーカス移動でレイアウト崩れが発生しない
- [ ] [補助] エラー表示後、正しい値に修正するとエラー表示が消える
